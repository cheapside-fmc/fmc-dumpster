FMC Route Format:

Header / Ending Example: ["LLBG","KSFO","CHP181",[...]]

This part of the format contains three main components, as described below:

  1. Departure airport: LLBG
  2. Arrival airport: KSFO
  3. Flight number: CHP181

Waypoints are contained within the inside set of square brackets, represented here with ellipses.

Waypoint Example: ["ARCHI",37.4908,-121.87554,"7000",false,"7000 FT"]

Waypoints each contain six fields which can be automatically filled up when making routes, or directly edited in post:

  1. Waypoint name: ARCHI
  2. Latitude: 37.4908
  3. Longitude: -121.87554
  4. Altitude: 7000
  5. Replace waypoint (Boolean): false
  6. Notes: 7000 FT

To summarize, the fields used in the FMC route format can be filled out as below:

FMC Field Allowed Values and Examples
Location Field Allowed Value Example
Header Departure airport 4 uppercase letters (ICAO code) LLBG
Arrival airport 4 uppdercase letters (ICAO code) KSFO
Flight number Text CHP181
Waypoint Waypoint name Text ARCHI
Latitude Number between -90 and 90 (-90 < lat < 90) 37.4908
Longitude Number between -180 and 180 (-180 < lon < 180) -121.87554
Altitude Multiple of 10 7000
Replace (replaces coordinates with database default) Boolean (true or false) false
Notes Text 7000 FT
